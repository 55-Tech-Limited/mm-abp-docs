## Prerequisites

You need an API key for the platform you want to use. Contact [support@55-tech.com](mailto:support@55-tech.com) to get your key.

| Platform | Auth Header | Base URL |
|----------|------------|----------|
| ABP (Automated Bet Placing) | `X-ABP-API-KEY` | `https://v2.55-tech.com` |
| MM (Market Making) | `X-API-Key` | `https://mmapi.55-tech.com` |

---

## ABP — Automated Bet Placing

### 1. Check connectivity

```bash
curl https://v2.55-tech.com/health
```

### 2. List your accounts

```bash
curl -H "X-ABP-API-KEY: YOUR_API_KEY" \
  https://v2.55-tech.com/accounts
```

### 3. Get a betslip (live odds)

```bash
curl -H "X-ABP-API-KEY: YOUR_API_KEY" \
  "https://v2.55-tech.com/betslip?fixtureId=ID&outcomeId=161&playerId=0"
```

### 4. Place an order

```bash
curl -X POST -H "X-ABP-API-KEY: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"orders": [{"fixtureId": "...", "outcomeId": 161, "playerId": 0, "stake": 10, "minPrice": 1.5}]}' \
  https://v2.55-tech.com/place-orders
```

### 5. Track results

```bash
curl -H "X-ABP-API-KEY: YOUR_API_KEY" \
  https://v2.55-tech.com/orders
```

## MM — Market Making

### 1. Check connectivity

```bash
curl https://mmapi.55-tech.com/health
```

### 2. Verify your identity

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/me
```

### 3. List your orders

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/orders
```

### 4. Check positions

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/positions
```

### 5. View profit & loss

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/pnl
```
