## 什么是 55-Tech？

55-Tech 运营两个核心交易平台：

- **做市（MM）** — 高频做市引擎，在预测市场交易所（Polymarket、Kalshi）上下单，并在博彩公司进行对冲
- **自动投注（ABP）** — 通过单一API在30多家博彩公司进行投注，提供实时跟踪和结算

两个平台均从 [OddsPapi](https://docs.oddspapi.io/) 获取实时赔率数据，并提供 REST + WebSocket API 供客户端集成。
