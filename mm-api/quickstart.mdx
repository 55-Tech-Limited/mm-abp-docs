<Note>
The MM engine places orders automatically based on bookmaker odds. The client API lets you **monitor** all trading activity and **control** your trading state (pause, resume, stop, start) and cancel orders. You do not place orders through this API — the engine does it for you.
</Note>

## Step 1: Verify your identity

Confirm your API key works and see your client info:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/me
```

Response:

```json
{
  "client": "your-client",
  "clientName": "Your Client Name",
  "oddsFormat": "decimal"
}
```

## Step 2: View exchange orders

See orders the engine has placed on exchanges:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/orders
```

Filter by exchange, fixture, or status:

```bash
# Filled Polymarket orders
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://mmapi.55-tech.com/api/v1/orders?exchange=polymarket&status=FILLED"

# Open orders for a specific fixture
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://mmapi.55-tech.com/api/v1/orders/open?fixtureId=id1000000861624412"
```

Each order includes exchange-specific identifiers, matched amounts, and odds snapshots:

```json
{
  "orderId": 123,
  "fixtureId": "id1000000861624412",
  "outcomeId": 161,
  "exchange": "polymarket",
  "exchangeOrderId": "0x1a2b3c...",
  "exchangeOutcomeId": "12345678",
  "orderStatus": "FILLED",
  "matchedStatus": "FULLY_MATCHED",
  "orderCents": 0.45,
  "orderStake": 100.0,
  "matchedStake": 100.0,
  "settlementStatus": "UNDECIDED",
  "bookmakerOutcomePrice": 1.808,
  "bookmakerOutcomeLimit": 5000.0,
  "createdAt": "2026-02-15T10:30:00Z",
  "matchedAt": "2026-02-15T10:30:05Z"
}
```

## Step 3: View hedge bets

See bets placed on bookmakers to hedge exchange fills:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/bets
```

Each bet is linked to an exchange order via `orderId`:

```json
{
  "betId": 456,
  "orderId": 123,
  "client": "your-client",
  "bookmaker": "vertex",
  "bookmakerBetId": "789",
  "placedPrice": 1.808,
  "placedStake": 100.0,
  "betStatus": "matched",
  "settlementStatus": null,
  "placedAt": "2026-02-15T10:30:08Z"
}
```

## Step 4: Check positions

View aggregated positions across all fixtures and exchanges:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/positions
```

Get a position summary:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/positions/summary
```

## Step 5: View profit & loss

Get your PnL breakdown:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/pnl
```

Filter by date range:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://mmapi.55-tech.com/api/v1/pnl?fromDate=2026-01-01&toDate=2026-01-31"
```

## Step 6: View fixtures

See which events the engine is trading on:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/fixtures
```

Filter by sport, tournament, or live status:

```bash
# Live fixtures only
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://mmapi.55-tech.com/api/v1/fixtures?live=true"

# Filter by tournament
curl -H "X-API-Key: YOUR_API_KEY" \
  "https://mmapi.55-tech.com/api/v1/fixtures?tournamentId=17"
```

## Step 7: List accounts

View your exchange accounts (credentials are never exposed):

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/accounts
```

```json
{
  "oddsFormat": "decimal",
  "accounts": [
    {
      "username": "0x1234...abcd",
      "exchange": "polymarket",
      "active": true,
      "maxOutcomeStake": 1000.0,
      "balance": 50000.0,
      "createdAt": "2026-01-15T12:00:00Z"
    }
  ],
  "count": 1
}
```

## Step 8: Trading controls

Check your current trading status:

```bash
curl -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/trading/status
```

Pause trading (keeps existing orders, stops new ones):

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/trading/pause
```

Resume after a pause:

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/trading/resume
```

Stop trading entirely (cancels all open orders):

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/trading/stop
```

Start trading from a stopped state:

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/trading/start
```

## Step 9: Cancel orders

Cancel all open orders at once:

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/orders/cancel-all
```

Cancel a specific order by ID:

```bash
curl -X POST -H "X-API-Key: YOUR_API_KEY" \
  https://mmapi.55-tech.com/api/v1/orders/123/cancel
```

## Step 10: Update client settings

Update your preferences (e.g. odds display format):

```bash
curl -X PATCH -H "X-API-Key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"oddsFormat": "decimal"}' \
  https://mmapi.55-tech.com/api/v1/clients/settings
```
